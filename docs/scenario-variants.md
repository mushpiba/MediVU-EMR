# 진료 환경별 확장 방향

현재 클릭형 데모는 내과 외래 한 증례에 집중한다. 이후 환경을 확장할 때 같은 화면을 단순 복제하지 않고, 임상 목표와 시간 압박에 맞춰 정보 우선순위를 바꾼다.

| 구분 | 내과 외래 데모 | 응급실 초진 | 호흡기내과 |
| --- | --- | --- | --- |
| 핵심 목표 | 대화와 종합 기록을 활용한 감별진단 | 중증도와 시간 민감 위험진단 탐지 | 호흡기 전문 검사와 치료 경로 결정 |
| 첫 화면 우선순위 | 대화, 키워드, 가능성순 Top 5 | 활력징후, 중증도, 즉시 배제할 위험진단 | 증상 궤적, 폐기능, 전문영상, 이전 치료 반응 |
| 후보 정렬 | 독립 추정 확률순 | 위험도와 긴급도 우선, 이후 가능성 | 질환군·표현형·프로토콜 적합도 중심 |
| 주요 데이터 | 과거 외래 기록, 기본 검사, 단순 영상 | 트리아지, 연속 활력징후, 현장검사, 투약·처치 시간 | 폐기능검사, 흉부 CT, 미생물 검사, 흡입제 이력 |
| 대표 행동 | 저장 외래 오더세트 선택 | 원클릭 응급 검사·처치 묶음과 재평가 타이머 | 전문 클리닉 프로토콜과 장기 추적 계획 |
| 경고 방식 | 근거 출처와 의료진 최종 판단 고지 | 위험진단을 확률과 무관하게 상단 고정 | 검사 품질·금기·전문 지침 버전 표시 |

## 응급실 초진 방향

- 진단 후보 위에 `즉시 확인` 영역을 두고 패혈증, 급성관상동맥증후군, 폐색전증처럼 놓치면 위험한 후보를 별도로 고정한다.
- 대화보다 활력징후 추세, 트리아지 등급, 마지막 처치 시각을 상단에 배치한다.
- 오더 화면은 검사·처치 묶음, 수행 상태, 재평가 타이머가 중심이 된다.
- 가능성이 낮더라도 위험도가 높은 후보를 숨기지 않으며, 확률만으로 우선순위를 설명하지 않는다.

## 호흡기내과 방향

- 같은 기침·호흡곤란 증례를 유지하되 폐기능검사, 흉부 CT, 객담검사, 흡입제 순응도를 추가한다.
- 진단 카드는 단순 질환명뿐 아니라 천식 표현형, COPD 악화 요인, 감염성·비감염성 폐침윤 분류를 지원한다.
- 진단별 태그는 학회 지침 버전, 병원 프로토콜, 기기 교육, 추적 검사 주기로 확장한다.
- 오더 화면에는 전문 검사 전제조건과 장기 추적 계획을 별도 섹션으로 둔다.

## 공통 원칙

- 실제 배포 전에 각 진료 환경의 의사 사용자로부터 정보 밀도, 위험 경고, 클릭 수를 검증한다.
- 확률·점수·위험도는 서로 다른 의미를 가지므로 화면에서 혼용하지 않는다.
- 원자료와 AI가 추출한 키워드를 분리하고, 의료진이 근거를 제외하거나 복원한 기록을 감사 가능하게 유지한다.
